# homebrew-tap

Homebrew tap for [gitusr](https://github.com/northwang-lucky/gitusr) - managed automatically by GoReleaser.

## Installation

```bash
brew tap northwang-lucky/tap
brew install gitusr
```

## Available Formulae

- `gitusr` - CLI tool for managing and switching between git user identities

This repository is automatically managed by [GoReleaser](https://goreleaser.com/). Please do not manually edit the Formula files.
